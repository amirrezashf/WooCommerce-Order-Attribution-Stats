# WooCommerce Order Attribution Stats

Displays detailed WooCommerce **Order Attribution** statistics grouped by **Origin**, including fixed and custom date ranges, pie charts, revenue analysis, average order value, HPOS compatibility, and 24-hour caching.

---

## Features

- 📊 Displays order attribution statistics based on **Origin**
- 🥧 Interactive pie chart for each reporting period
- 💰 Displays total sales amount
- 📦 Displays total order count
- 📈 Displays average order value
- 📅 Built-in reports for:
  - Last 30 days
  - Last 90 days
  - Last 180 days
  - Last 360 days
- 🗓️ Custom date range reporting
- ⚡ 24-hour caching for better performance
- 🔄 Manual cache refresh button
- 🚀 Supports both WooCommerce Classic Orders and HPOS
- 🎨 Modern and responsive WordPress admin interface
- 🔒 Read-only reporting (does not modify any WooCommerce data)

---

## Requirements

- WordPress 6.0+
- WooCommerce 8.0+
- PHP 7.4+

---

## Installation

1. Download this repository.
2. Upload the plugin to:

```
/wp-content/plugins/
```

3. Activate the plugin from **Plugins**.
4. Open:

```
WordPress Admin → Order Attribution Stats
```

# فارسی

افزونه‌ای برای نمایش آمار **Order Attribution** ووکامرس بر اساس **Origin** همراه با گزارش‌های آماری، نمودار دایره‌ای، مبلغ فروش، میانگین سفارش و بازه‌های زمانی مختلف.

- نمایش نمودار دایره‌ای
- نمایش تعداد سفارش‌ها
